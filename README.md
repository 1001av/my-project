# Synthetic Data Generator with LangChain & LangGraph

## Summary
This project implements a **Python-based synthetic data generation system** that creates realistic, synthetic data for SQL database tables using **LangChain** and **LangGraph**.

The application connects to a SQL database, reads table schemas, generates synthetic records using LLM-driven prompt workflows, and inserts the generated data back into the database. The project follows **modular architecture, clean coding standards, and industry best practices**.
